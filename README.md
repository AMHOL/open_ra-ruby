# Openra Rubygem


### Installation
```sh
gem install openra
```

NOTE: Requires Ruby ([Installation guide](https://www.ruby-lang.org/en/documentation/installation/))

### Updating
```sh
gem update openra
```

### Example usage

```sh
openra replay-data /payh/to/replay.orarep [--format json|pretty-json|yaml]
```
